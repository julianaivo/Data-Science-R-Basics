
# 📊 Data Science with R — Harvard University

Personal study repository for the introductory **Data Science: R
Basics** course from Harvard University.

This course focuses on building a foundation in R programming and
learning how to wrangle, analyze, and visualize data using real-world
datasets.

## 📚 Topics Covered

- Basic R syntax
- Data types and vectors
- Vector arithmetic and indexing
- Sorting and operations in R
- Conditional statements and loops
- Data wrangling with `dplyr`
- Data visualization with `ggplot2`

## 👨‍🏫 Instructor

- Rafael Irizarry  
  Professor of Biostatistics at Harvard T.H. Chan School of Public
  Health

## 🎯 About the Course

The course is part of Harvard’s Professional Certificate Program in Data
Science and introduces core R programming concepts through practical
data analysis exercises.
